Mysql

Library for accessing Mysql database

Connection

Establish a connection to the database 

	'connection connect( {
			host => string,
			port => int,
			user => string,
			pass => string,
			socket => string?
	} );
	void close( 'connection c );
	void selectDB( 'connection c, string db );
	'result request( 'connection c, string sql );

Result

A request result 

	int result_get_length( 'result r );
	int result_get_nfields( 'result r );
	object result_next( 'result r );
	string? result_get( 'result r, int n );
	int? result_get_int( 'result r, int n );
	float? result_get_float( 'resulr r, int n );
