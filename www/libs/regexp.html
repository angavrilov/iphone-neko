Regexp

Regular expressions using PCRE engine.

'regexp regexp_new(string) Build a new regexp bool regexp_match('regexp, string, pos : int, len : int) Match len chars of a string starting at pos using the regexp. Return true if match found string regexp_replace('regexp, from : string, by : string) Perform a replacement using a regexp string regexp_replace_all('regexp, from : string, by : string) Perform a replacement of all matched substrings using a regexp string regexp_matched('regexp, n : int) Return the nth matched block by the regexp. If n is 0 then return the whole matched substring { pos => int, len => int} regexp_matched_pos('regexp, n : int) Return the nth matched block position by the regexp. If n is 0 then return the whole matched substring position
